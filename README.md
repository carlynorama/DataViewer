#  DataViewer

This project is a simple data viewer and curve fitter currently intended for small datasets in house. 

[![Swift Version][swift-image]][swift-url]

## Installation

Requires:
   -  X-Code 14 or later
   -  [DataHelper package](https://github.com/carlynorama/DataHelper)
